# Employee Attendance Analytics Dashboard

Dashboard analitik kehadiran karyawan berbasis Python dan Streamlit untuk membantu proses monitoring serta visualisasi data absensi secara interaktif.

## Project Overview

Project ini dikembangkan untuk mengolah dan memvisualisasikan data absensi karyawan sehingga informasi kehadiran dapat dipantau dengan lebih cepat dan mudah.

Dashboard mampu menampilkan statistik kehadiran, tingkat keterlambatan, distribusi status absensi, serta peringkat karyawan dengan tingkat kehadiran terbaik.

## Features

* Upload file Excel (.xlsx)
* Filter berdasarkan karyawan
* KPI Kehadiran
* Attendance Rate
* Late Rate
* Pie Chart Distribusi Kehadiran
* Top Attendance Employee Chart
* Top Performer Ranking
* Insight Otomatis

## Technologies

* Python
* Pandas
* Streamlit
* Matplotlib
* OpenPyXL

## Dataset

Dataset terdiri dari data absensi karyawan yang memuat:

* Nama Karyawan
* Tanggal
* Status Kehadiran

Status yang digunakan:

* Hadir
* Terlambat
* Izin
* Sakit
* Alpha

## How to Run

Install dependencies:

pip install -r requirements.txt

Run dashboard:

streamlit run app.py
